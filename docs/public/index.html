Tachyon

Tachyon is a 685 byte javascript module that improves the user experience of your website by making your links feel virtually instant.

To test if Tachyon would’ve helped,

Tachyon uses browsers’ built in features to markedly improve the speed of your website, resulting in a better user experience and potentially even significantly more conversions.

Using Tachyon

Deploying Tachyon to your site is extremely easy. For most people the steps will look like this:

  1. Copy one of the <script></script> tags in the section above.

  2. Add it between your website’s <head></head> tags.

  3. Done! Your website will now feel blazingly fast.

In most cases, this is all you need to do to implement Tachyon. For more advanced users, Tachyon can be downloaded here for use without a CDN; it is also available via npm as tachyonjs.

Technical Details

Tachyon’s repository is available here.

Tachyon leverages built-in browser features: it automatically generates <link rel="prefetch" href="___"> tags to preload content when a user hovers their cursor on any <a href="___"></a> tag for more than 50ms. The browser then begins downloading and caching the page content of href="___", so when the user clicks through the anchor, the page is already cached and ready to be loaded into the frame.

These prefetch requests are given extremely low priority; they will not interfere with the speed of other requests.

Tachyon & Alternatives

Tachyon could be considered a light-weight alternative to projects like instant.page, which offer a similar feature set. That being said, Tachyon has some serious advantages compared to other similar projects.

To get an idea at a glance, Tachyon improves your website’s speed 2-5x more and is 4.3x smaller compared to instant.page.

At 685 bytes minified, Tachyon has a major advantage compared to alternatives; Tachyon is 4.3x smaller than instant.page, leading to the same performance boost with considerably less overhead.

Tachyon uses a standard timing of 50ms from initial mouse hover to initializing the prefetch, though this will be configurable in the future. This timing is a happy medium between both preserving bandwidth and speed.

Many privacy extensions disable the functionality that Tachyon relies on. Please keep this in mind as you test your site’s Tachyon implementation.
