Tachyon is an 870 byte javascript file that improves the speed and user experience of your website by making your links instant.

Tachyon is a very simple implementation of a page prediction system. It makes the user's browser download linked content before they even click a link by taking advantage of the time a user hesitates before clicking; when the user clicks the link, the page is already downloaded and is displayed instantly.

To test if Tachyon would've helped, click me! No! Yes! (Loaded for )Remove cursor before trying again.

This incredibly simple technique markedly improves the speed of your website, resulting in a better user experience and potentially even translates to significantly more conversions.

| Using Tachyon

Deploying Tachyon to your site is extremely easy.
For most people the steps will look like this:

  1. Copy one of the <script></script> tags in the section above.

  2. Add it between your website's <head></head> tags.

  3. Done! Your website will now feel blazingly fast.
In most cases, this is all you need to do to implement Tachyon. For more advanced users, Tachyon can be downloaded here for use without a CDN; it is also available via npm as tachyon-npm.

| Technical Details

Tachyon uses built-in browser features: it automatically generates <link rel="prefetch" href="⠀ ⠀"> tags to preload content when a user hovers their cursor on any <a href="⠀ ⠀"></a> tag for more than 50ms. The browser then begins downloading and caching the page content of href="⠀ ⠀", so when the user clicks through the anchor, the page is already cached and ready to be loaded into the frame.

These prefetch requests are given extremely low priority; they will not interfere with the speed of other requests.

Tachyon could be considered a light-weight alternative to projects like instant.page, which offer a similar feature set. At 870 bytes minified, Tachyon has a major advantage compared to alternatives; Tachyon is 4.27x smaller than instant.page, leading to the same performance boost with considerably less overhead.

In order to achieve this, Tachyon has a slightly reduced scope, with no whitelist feature or configurability out of the box. Tachyon is only 49 extremely readable lines, though, meaning extensibility is a breeze. Both of these features are planned, and will be implemented with the same light-weight philosophy.

Tachyon uses a standard timing of 50ms from initial mouse hover to initializing the prefetch, though this will be configurable in the future. This timing is a happy medium between both preserving bandwidth and speed.

Many privacy extensions disable the functionality that Tachyon relies on. Please keep this in mind as you test your site's Tachyon implementation.
